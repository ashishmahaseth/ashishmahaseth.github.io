# Ashish Mahaseth — Portfolio

A dark, terminal-aesthetic single-page portfolio website built with Tailwind CSS, AOS animations, and carefully crafted glassmorphism + glow effects.

## 🌐 Live Demo

Deploy this site via GitHub Pages:

1. Go to your repository **Settings → Pages** (or **Settings → Code and automation → Pages** on the new UI).
2. Under **Build and deployment → Source**, select **Deploy from a branch**.
3. Choose the branch (e.g. `main`) and root (`/`) folder.
4. Click **Save**. GitHub Pages will build and serve the site.

Your portfolio will be available at:

```
https://<YOUR-USERNAME>.github.io/<REPO-NAME>/
```

## 🛠 Tech Stack

- **Tailwind CSS** (v3 via CDN)
- **AOS** (Animate On Scroll)
- **Boxicons** + **Material Symbols**
- **Google Fonts** — Space Grotesk + IBM Plex Mono
